Style

Some colors and animations are available for the sidebar.

Colors

The sidebar comes with some color options:

Color Description
default Uses the default element colors (starting point for customization).
dark Applied using the class .sidebar-dark.
light Applied using the class .sidebar-light.

Animations

The sidebar comes with two animations:

Animation Description
.sidebar-fade Fade in and fade out effects.
.sidebar-slide-left Slide from left to right.
