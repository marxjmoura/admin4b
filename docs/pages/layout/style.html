Style

Some colors and animations are available for the sidebar.

Colors

The sidebar comes with some color options:

Color Description
default Uses the default element colors (starting point for customization).
dark Applied using the class .sidebar-dark.
light Applied using the class .sidebar-light.

Examples

<div class="app-sidebar"></div>
<div class="app-sidebar sidebar-dark"></div>
<div class="app-sidebar sidebar-light"></div>

Animations

The sidebar comes with two animations:

Animation Description
.sidebar-fade Fade in and fade out effects.
.sidebar-slide-left Slide from left to right.

Examples 

<div class="app-sidebar sidebar-fade"></div>
<div class="app-sidebar sidebar-slide-left"></div>

Width

The width of sidebar can only be changed by code. See the example below: 

/* The media query prevents breaking the sidebar on smaller screens */
@media (min-width: 768px)
{
  .app .app-body .app-sidebar
  {
    width: 360px; /* Your custom width */
  }
}
